# WeiNews - 微新闻
本 App 提供每日新闻阅读服务。新闻资源来自@聚合数据免费Api，本人写的一个小案例。
## 引用库
- RxJava
- RxAndroid
- OKHttp
- Retrofit
- Glide
- Gson
- ButterKnife
- TextPathView
- Jsoup
- EventBus
## 应用设计
- 引用 TextPathView 来制作欢迎页面，动态路径绘制动画.
- Retrofit + OKHttp + RxJava 请求网络
- MVP 程序框架，对 MVP 的封装和使用
- ViewPager + Fragment 的懒加载功能
- RecylerView 的多布局和下拉刷新
- 白天/夜间模式的切换功能
